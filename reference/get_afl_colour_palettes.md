# Returns a table with the colour palettes for all teams

`get_afl_colour_palettes` returns a data frame containing the AFL team's
primary, secondary and tertiary colours as applicable The data for this
function is hosted on github.

## Usage

``` r
get_afl_colour_palettes()
```

## Value

a data table containing team long name, team abbreviation, and colours

## Examples

``` r
if (FALSE) { # \dontrun{
# Gets all data
get_afl_colour_palettes()
} # }
```
