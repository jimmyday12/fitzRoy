# Internal function to ensure venue names match between different sources and also name changes across time. This gets applied to any web scraper, transforming all of them to AFL Tables naming conventions.

Internal function to ensure venue names match between different sources
and also name changes across time. This gets applied to any web scraper,
transforming all of them to AFL Tables naming conventions.

## Usage

``` r
replace_venues(venue)
```

## Arguments

- venue:

  Venue name
