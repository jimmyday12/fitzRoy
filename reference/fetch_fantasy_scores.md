# Fetch AFL Fantasy (Dream Team) Scores

Fetch AFL Fantasy (Dream Team) Scores

## Usage

``` r
fetch_fantasy_scores(year = 2025, rounds = 1:30)
```

## Arguments

- year:

  Integer. AFL season year.

- rounds:

  Integer vector. Rounds to fetch (default: 1:30).

## Value

A data frame of Dream Team scores.
