# Fetch AFL All-Australian Team or Squad

Fetch AFL All-Australian Team or Squad

## Usage

``` r
fetch_awards_allaustralian(season, type = c("team", "squad"))
```

## Arguments

- season:

  A single year (e.g., 2023)

- type:

  Either "team" (final 22) or "squad" (initial 44)

## Value

A tibble with player/team info
